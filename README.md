# La Stellina — Luxury Italian Restaurant Website

A fully responsive concept website for a fine dining Italian restaurant.

## Features
- Bilingual English / Italian language toggle
- Interactive tabbed menu system
- Reservation form
- Smooth scroll animations
- Fully responsive — mobile, tablet and desktop
- Elegant Italian-themed design

## Technologies Used
HTML5 · CSS3 · Vanilla JavaScript · Google Fonts
